## 📖 Sobre

O **LumenSlide** é um projeto **aberto e colaborativo** para gerar slides litúrgicos da Santa Missa de forma **dinâmica**, **acessível** e **gratuita** — pensado para projeção em paróquias, comunidades e capelas.

A proposta é simples: que qualquer comunidade católica possa, a partir do calendário litúrgico do dia, montar e projetar uma celebração com leituras, salmos, cantos e o ordinário da Missa em poucos cliques.

## 🎯 Funcionalidades planejadas

- **Leituras do dia (lecionário)** — slides automáticos com 1ª leitura, salmo responsorial, 2ª leitura e evangelho conforme o calendário litúrgico.
- **Ordinário da Missa** — partes fixas (Ato Penitencial, Glória, Credo, Santo, Cordeiro de Deus etc.) com ritos próprios para cada tempo.
- **Cantos / hinário** — banco de cantos litúrgicos com letras paginadas em slides; possibilidade de upload de cantos próprios da comunidade.
- **Calendário litúrgico** — identificação automática do tempo litúrgico (Advento, Natal, Quaresma, Páscoa, Tempo Comum) e da cor litúrgica do dia.
- **Modo apresentação** — projeção em tela cheia com navegação por teclado, controle remoto e atalhos.
- **Temas visuais** — temas claro/escuro e variações por tempo litúrgico (roxo da Quaresma, branco da Páscoa, verde do Tempo Comum etc.).
- **Exportação** — gerar PDF ou PPTX da celebração para uso offline.
- **PWA** — uso offline em paróquias com internet limitada.

## 🚀 Começando

### Pré-requisitos

- [Node.js](https://nodejs.org) 20+
- [pnpm](https://pnpm.io), `npm` ou `yarn`

### Instalação

```bash
git clone https://github.com/SEU-USUARIO/lumenslide.git
cd lumenslide
npm install
npm run dev
```

A aplicação ficará disponível em `http://localhost:5173`.

### Build de produção

```bash
npm run build
npm run preview
```

## 🧱 Stack

- **React 18** + **TypeScript**
- **Vite** (build / dev server)
- **CSS Modules** (sem dependência de UI kit pesado)
- **Vitest** (testes — futuro)
