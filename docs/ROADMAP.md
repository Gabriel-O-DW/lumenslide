# Roadmap — LumenSlide

Este roadmap é um documento vivo. Sugestões e PRs são bem-vindos!

## v0.1 — Fundação (MVP)

- [x] Scaffold React + Vite + TypeScript
- [x] Estrutura de tipos para slides litúrgicos (`Leitura`, `Salmo`, `Canto`, `Ordinario`, `Generico`)
- [x] Visualizador de slides com navegação por teclado (← →, Espaço, Esc)
- [x] Modo apresentação em tela cheia
- [x] Tema claro/escuro
- [ ] Persistência local da celebração montada (localStorage)

## v0.2 — Calendário litúrgico

- [ ] Cálculo do tempo litúrgico do dia (Advento, Natal, Quaresma, Páscoa, Comum)
- [ ] Cor litúrgica automática
- [ ] Festa / memória / solenidade do dia

## v0.3 — Lecionário

- [ ] Estrutura de dados para leituras (citação + texto)
- [ ] Importação de leituras a partir de fonte externa configurável
- [ ] Slide de leitura com citação no rodapé

## v0.4 — Cantos

- [ ] Banco de cantos (JSON)
- [ ] Editor de canto (refrão / estrofes)
- [ ] Auto-pagina: divide canto longo em vários slides

## v0.5 — Editor de celebração

- [ ] Drag & drop de blocos litúrgicos
- [ ] Templates por tempo litúrgico
- [ ] Reordenação e duplicação de slides

## v0.6 — Exportação e PWA

- [ ] Export PDF
- [ ] Export PPTX
- [ ] PWA (uso offline)

## Ideias futuras

- Integração com controle remoto via celular (mesma rede local)
- Modo "duas telas": apresentador vê próxima leitura / nota
- Suporte multi-idioma (i18n)
- Modo "comunidade pequena": fonte grande, tela única
